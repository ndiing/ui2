# MDButton

A custom element representing an icon button.

## Instance properties

| name | type   | default      | description                   |
| ---- | ------ | ------------ | ----------------------------- |
| icon | String |              | The icon of the button        |
| type | String | icon-button  | The type of the button        |
| ui   | String |              | The UI style of the button    |
