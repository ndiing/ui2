# MDSlider Component

## Element

Dalam penggunaan sehari-hari, MDSlider digunakan sebagai elemen `md-slider`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
