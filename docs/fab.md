# MDFab

A custom element representing a FAB (Floating Action Button).

## Instance properties

| name | type   | default | description                   |
| ---- | ------ | ------- | ----------------------------- |
| icon | String |         | The icon of the FAB           |
| ui   | String |         | The UI styles of the FAB      |
