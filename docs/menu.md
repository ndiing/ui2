# MDMenu Component

## Element

Dalam penggunaan sehari-hari, MDMenu digunakan sebagai elemen `md-menu`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
