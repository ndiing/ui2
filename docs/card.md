# MDCard Component

## Element

Dalam penggunaan sehari-hari, MDCard digunakan sebagai elemen `md-card`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
