# MDButton Component

## Element

Dalam penggunaan sehari-hari, MDButton digunakan sebagai elemen `md-button`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
