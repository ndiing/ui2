# MDBadge

A custom element representing a badge.

## Instance properties

| name | type    | default | description        |
| ---- | ------- | ------- | ------------------ |
| label| Number  |         | The label of the badge |
| max  | Number  | 999     | The maximum value to display, with exceeding values being represented as 'max+' |
