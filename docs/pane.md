# MDPane Component

## Element

Dalam penggunaan sehari-hari, MDPane digunakan sebagai elemen `md-pane`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
