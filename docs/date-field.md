# MDDateField Component

## Element

Dalam penggunaan sehari-hari, MDDateField digunakan sebagai elemen `md-date-field`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
