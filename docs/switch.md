# MDSwitch Component

## Element

Dalam penggunaan sehari-hari, MDSwitch digunakan sebagai elemen `md-switch`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
