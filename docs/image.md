# MDImage Component

## Element

Dalam penggunaan sehari-hari, MDImage digunakan sebagai elemen `md-image`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
