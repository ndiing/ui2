# MDForm Component

## Element

Dalam penggunaan sehari-hari, MDForm digunakan sebagai elemen `md-form`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
