# MDDialog Component

## Element

Dalam penggunaan sehari-hari, MDDialog digunakan sebagai elemen `md-dialog`. Perhatikan bahwa elemen kustom seperti ini mungkin tidak memerlukan tag penutup terpisah seperti elemen HTML standar.
